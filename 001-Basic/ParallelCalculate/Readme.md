

# Parallel For Each

## Parallel For Each (Sync)

* 作法
    1. 使用 `Parallel.For`
    2. 使用 `Parallel.ForEach`
    3. 使用 `Plinq`



---

## Parallel For Each (Async)

* 作法
    1. 使用 `DataFlow`
    2. 使用 `Parallel.ForEachAsync`
        * **限 .NET 6 以後的專案**
